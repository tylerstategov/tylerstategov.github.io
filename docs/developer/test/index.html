Basic Auth

My development Azure system has BasicAuth enabled. Any new system will not have BasicAuth enabled. These next notes may not be for you.

Get-OrganizationConfig | Format-Table Name,OAuth* -Auto

Set-OrganizationConfig -OAuth2ClientProfileEnabled $true
Set-OrganizationConfig -OAuth2ClientProfileEnabled $false

Set-OrganizationConfig -DefaultAuthenticationPolicy <policyname>
Set-OrganizationConfig -DefaultAuthenticationPolicy $null

Get-OrganizationConfig

Set-User -Identity <UserIdentity> -AuthenticationPolicy <PolicyIdentity>

Get-User -ResultSize Unlimited | select Name,AuthenticationPolicy
Get-User | Select Name,AuthenticationPolicy

Connect-ExchangeOnline -UserPrincipalName navin@contoso.com
New-AuthenticationPolicy -Name "GovCrm Block Basic Auth"
Set-User -Identity laura@contoso.com -AuthenticationPolicy "GovCrm Block Basic Auth"
Get-AuthenticationPolicy -Identity "GovCrm Block Basic Auth"

Connect-ExchangeOnline -UserPrincipalName navin@contoso.com
Get-AuthenticationPolicy
Set-OrganizationConfig -DefaultAuthenticationPolicy "GovCrm Block Basic Auth"
Remove-AuthenticationPolicy -Identity "GovCrm Block Basic Auth"
