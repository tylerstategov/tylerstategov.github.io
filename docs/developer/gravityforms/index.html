Gravity Forms

Website

Constituents will submit comments or requests to the Governor’s website, https://governor.hawaii.gov/contact-us/.

The Contact Us webpage contains several tiles. The number of tiles will vary, depending on the needs of the office. The WebJob is concerned with only some tiles. Clicking on a tile will open a WordPress Page containing a Gravity Forms form.

Gravity Forms

The Governor’s website Contact Us webpage has multiple tiles with links to Gravity Forms forms. Clicking on a tile will open a WordPress Page containing a Gravity Forms form.

Here are the Gravity Forms on the WordPress administration page. There are two pages of forms. If a form is toggled green , then it is active.

Do not delete an inactive form if it has data entries.

Gravity Form XML Notification

If an active form has an XML notification, then we need WebJob code to process the XML.

The weirdo Notification name “XML To IQ GFUEANZ” was a requirement of an older version of Gravity Forms. The newer version of Gravity Forms does not require the “GFUEANZ” to email the attachment files in the Notification. You may rename the Notification.

When the constituent submits a form, Gravity Forms will send an email to officeofthegovernor@hawaii.gov. The email contains XML data.

Exchange

The WebJob uses the Microsoft Exchange Web Services to sign-in to the officeofthegovernor@hawaii.gov email account.

Job Schedule

The project contains file Settings.job. The content of this file is a JSON formatted crontab. Here is the content:

{
"schedule": "0 0 7-20 * * 1-6"
}

This is a cron schedule. Note: this cron schedule has a seconds value.

The WebJob will execute at the times listed in the cron schedule.

Config

The application configuration values are kept in the Azure configuration section and project file app.config.
