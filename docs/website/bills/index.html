Acts and Vetoes

New Bills, Acts, and Vetoes pages are created once a year before session starts.

  • https://governor.hawaii.gov/2020-bills/
  • https://governor.hawaii.gov/2020-bills/2020-acts/
  • https://governor.hawaii.gov/2020-bills/2020-vetoes/

This is an image of the 2020 Bills page. The following instructions will explain how to add a new 2021 Bills page.

Create two new WordPress categories

First create 2 new categories. These new categories will be added to the new pages we create later.

Open the Categories page.

This is a list of the current categories. You must add 2 new categories, 2021 Acts and 2021 Vetoes.

To create the new 2021 Acts category, fill out the Name and Slug fields. See the yellow-highlighted fields below. The name is how it appears on the site. The name may contain spaces. The “slug” is the URL-friendly version of the name. It is usually all lowercase and contains only letters, numbers, and hyphens, NO SPACES.

Create the 2021 Vetoes category. Follow the steps above, changing “acts” to “vetoes”.

Here is an image of the new categories. Remember the ID’s of the new categories. We’ll need the ID’s when we create the 2021 pages (next steps). In this example, the 2021 Acts ID is 105 and the 2021 Vetoes ID is 106.

Create three new WordPress pages

We must create 3 new WordPress pages.

  1. 2021 Bills
  2. 2021 Acts
  3. 2021 Vetoes

1. Create the Bills page

First create the 2021 Bills page. Use the 2020 Bills page as a template for the 2021 Bills page. Copy and paste the 2020 HTML code to create the new 2021 page.

Click the left-menu item to open the list of WordPress pages. There are over 100 pages. Click the next-page button (highlighted yellow below) until you see 2020 Bills in the list. Click on the 2020 Bills title.

After clicking the title name, the “Edit Page” is opened and the page details are displayed. Copy the 2020 Bills HTML code highlighted yellow below.

To see the HTML code, set the viewing mode to “Text” mode instead of “Visual” mode.

The following is the 2020 HTML code.

<ul>
<li><h4><a href="https://governor.hawaii.gov/2020-bills/2020-acts/">Acts</a></h4></li>
<li><h4><a href="https://www.capitol.hawaii.gov/advreports/advreport.aspx?year=2020&amp;report=deadline&amp;rpt_type=gov_signed&amp;measuretype=HB,SB&amp;title=Bills%20signed%20by%20the%20Governor" target="_blank" rel="noopener noreferrer">Bills signed by the Governor</a></h4></li>
<li><h4><a href="https://governor.hawaii.gov/2020-bills/2020-vetoes/">Vetoes</a></h4></li>
</ul>

Create a new page by clicking the Add New button or click the Pages Add New menu item.

 or 

The Add New Page window is opened.

  • Enter the title “2020 Bills” into the title box.
  • Enter the 2020 HTML code in the editor box. The HTML is the code you copied in a previous step.
  • Update the code in the editor box. Change all instances of 2020 to 2021. Change only complete instances of 2020.
  • Do NOT change any %20. If you change any stand-alone 20, the hyperlink URL will be incorrect.

A URL is not allowed to have spaces, so a special code, %20, was invented by the internet inventors. It’s all magic.

Click the Publish button to save the page and make the page available to the public.

This is the final 2021 Bills page.

2. Create the Acts page

After creating the 2021 Bills page, create the 2021 Acts page. Use the 2020 Acts page as a template for the 2021 Acts page. Copy and paste the 2020 Acts HTML code to create the new 2021 page.

Open the list of WordPress pages. Find the 2020 Acts page. The title of the Acts page has a leading em-dash. Click on the 2020 Acts title.

After clicking the title, the “Edit Page” is opened and the page details are displayed. Copy the 2020 Acts HTML code highlighted yellow below.

Click the Add New button.

Add News navigation menu item

In the News page, https://governor.hawaii.gov/category/newsroom/, the right-side navigation menu has the Bills listed by year. Each year when session begins, we add a new menu option for the new year.

After creating the 2021 Bills, 2021 Acts and 2021 Vetoes pages, we add a menu item to the right-side Newsroom navigation menu.

To add to the Newsroom right-side menu, select the Appearance/Menus option.

  • Select a menu to edit from the dropdown: Newsroom.
  • Select the checkbox for page 2021 Bills.
  • Click the Add to Menu button.

After clicking the Add to Menu button, the new 2021 Bills menu item appears at the bottom of the menu structure.

“Drag and drop” the 2021 Bills menu option to the top.

Now that the 2021 Bills menu option is in place, click the Save Menu button.

The new Newsroom right-side navigation menu will look like this:

Post new acts and vetoes

During Session, post new acts and vetoes. Look at a 2020 post to see the content format.

There are multiple ways to start a new post.

Start a new post. Be sure to select a category from the category list on the right.

If you forget to select a category, the post will appear on the main news page. To fix the missing category problem, edit the post, unselect the Main category and select the correct Act or Veto category.
