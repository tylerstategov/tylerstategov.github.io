Contact Us

This will show how to add or remove tiles from the Contact Us page.

Add tile

  • Open the Contact Us edit page.
  • Add code for new tile.
  • Enable the tile’s linked-to page.

Remove tile

  • Open the Contact Us edit page.
  • Delete the tile code.
  • Disable the tile’s linked-to page.

Contact us

Constituents submit comments or requests using the Contact Us page of the Governor’s website, https://governor.hawaii.gov/contact-us/.

The Contact Us webpage contains several tiles. The number of tiles will vary, depending on the needs of the office. This image shows 9 tiles. Sometimes there are 8 tiles, other times there are 10 tiles.

WordPress pages

Open WordPress. Click the left-menu item to open the list of WordPress pages. There are over 100 pages. Click the image of next page button next-page button (highlighted yellow below) until you see Contact Us in the list.

To add or remove a tile, we edit the Contact Us page. Click the Contact Us title to open the editing page. The page ID is 14.

Enable the linked page

If clicking a tile opens a WordPress page, you must change the page’s visibility to public. If you fail to change the visibility, you’ll get a 404 page not found error.

The page visibility value determines if a page may be viewed by the public. The next image shows the page visibility is set to private. If a page’s visibility is set to private, the public cannot see the page and will get a 404 page not found error.


To make a page available to the public, set the page’s visibility to public. Click Edit.

Click Edit.

Check Public.

Click OK.

Click Update.

The page visibility is now set to public. The public can see the page.

Disable the linked page after deleting tile

After deleting a tile, you may have to disable the linked-to page. The links are out in the wild and constituents have bookmarked the link and will navigate to the page. To disable a page, set the linked-to page visibility to private.

Click Edit.

Click Private.

Click OK.

Click Update.

The page visibility is now set to private. The public cannot view the page.
